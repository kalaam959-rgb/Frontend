# Virtual Campus

A React + Three.js prototype for an AI-powered virtual college campus and student portal.

## Included experience
- 3D campus overview with building-focused navigation
- Student portal dashboard with attendance and results preview
- Persistent AI chat widget for navigation and campus guidance
- Department tour placeholder route and responsive layout

## Run locally
1. Install dependencies with `npm install`
2. Start the dev server with `npm run dev`
3. Open the local Vite URL shown in the terminal

## Notes
- The app uses React Router, Tailwind CSS, Three.js, and React Three Fiber.
- The 3D experience is intentionally lightweight and can be expanded into a full guided-tour system and authenticated portal later.
